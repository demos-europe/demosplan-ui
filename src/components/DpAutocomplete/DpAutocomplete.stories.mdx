# Autocomplete

DpAutocomplete is a wrapper for https://npm.io/package/vue-omnibox which is a simple search box for vue applications
that is able to auto complete search queries.

export const Template = (args, { argTypes }) => ({
  components: { DpAutocomplete },
  template: `<dp-autocomplete v-bind="$props" placeholder="placeholder" />`
});

## Default usage

```html
<dp-autocomplete
  name="search"
  id="autocomplete_search"
  v-model="currentAutocompleteSearch"
  :route-generator="(searchString) => {
    return Routing.generate('API-endpoint', {
      maxResults: 9,
      query: searchString
    })
  }"
  :options="autocompleteOptions"
  placeholder="placeholder text"
  label="value" />
```
