# ButtonIcon

The ButtonIcon component is used to add a button with an icon that can be selected depending on the position.

The text inside the button is hidden, but the tooltip text can be specified.

## Default usage

```html
<dp-button-icon
  icon="fa fa-check"
  text="Tooltip-Text" />
```
