# Card

The DpCard component used to add Card section with `heading`, `heading-tooltip` props, and include some text or component
like `dp-button` or `dp-loading`.

## Default usage

```html
<dp-card
    :heading="Translator.trans('heading')"
    :heading-tooltip="Translator.trans('analytics.visitorMetrics.contextualHelp')">
    Content goes here (into the default slot).
    Cards typically also contain actionable controls like buttons:
    <div class="text-right u-mt">
      <dp-button
        :href="userFilteredSegmentUrl"
        :text="Translator.trans('tasks.view')" />
    </div>
</dp-card>
```
