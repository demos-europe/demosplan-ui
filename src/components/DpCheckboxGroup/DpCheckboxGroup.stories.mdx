# CheckboxGroup

The `CheckboxGroup` component renders an inputs group of type `checkbox`.

Use it for a multiple options that the user can turn on or off.

## Default usage

```html
<dp-checkbox-group
  :label="label"
  :options="options"
  @update="checked => someFunction(checked)"
  :selected-options="someFunction(options)" />
```
