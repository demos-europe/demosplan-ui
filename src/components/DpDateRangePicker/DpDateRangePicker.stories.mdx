# DateRangePicker

The DateRangePicker component allows users to enter two dates within a given range either by typing
or by selecting two dates from the calendar.

## Default usage

```html
<dp-date-range-picker
    start-id="startDate_id"
    start-value="startValue"
    end-id="endDate_id"
    end-value="endValue" />
```
