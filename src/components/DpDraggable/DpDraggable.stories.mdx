# Draggable

Use the Draggable component to change the order of items within a list.


## Default usage

```html
<dp-draggable
    v-model="currentList"
    :opts="draggableOptions">
    <div class="o-sortablelist__item u-pv-0_5 u-pl-0_5 border--top">
      <p>Content 1</p>
    </div>
    <div class="o-sortablelist__item u-pv-0_5 u-pl-0_5 border--top">
      <p>Content 2</p>
    </div>
</dp-draggable>
```
