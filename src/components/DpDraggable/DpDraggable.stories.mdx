# Draggable

Use the Draggable component to change the order of items within a list.


## Default usage

```html
<dp-draggable
    v-model="currentList"
    :opts="draggableOptions">
    <div
        v-for="item in currentList"
        :key="item.id">
        {{ item.name }}
    </div>
</dp-draggable>
```
