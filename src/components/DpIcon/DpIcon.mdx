# Icon

The `DpIcon` component renders SVG icons, exposing a selected set of [PhosphorIcons](https://phosphoricons.com/) in regular variants
and some icons in filled variants.

### Xmark

Use for these cases:

* In tables, it can indicate success vs. error in columns with limited space in tandem with the "check" icon
* As a button icon, it is used to cancel an action
* In form fields, it can be the trigger to clear the field

Do <strong>not use</strong> the xmark icon for destructive actions. Use [`delete`](#delete) instead.

  name="Xmark"
