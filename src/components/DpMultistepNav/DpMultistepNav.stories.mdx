# Multistep navigation

The Multistep navigation generates a horizontal bar with n steps. The steps can be decorated with a fa-icon.

Clicking on a step emits the new step index.

## Default usage

```html
<dp-multistep-nav
  @change-step="val => step = val"
  :active-step="step"
  :steps="[{
       label: 'my text here',
       icon: 'fa-some-icon',
       title: 'In this step you can do this'
      }, {
       label: 'this is the second step',
       icon: 'fa-some-other-icon',
       title: 'In this step you can do that'
      }, {
       label: 'and so on'
      }]" />
```
