# Notification

The Notification component used to display brief messages or alerts to users.

These messages are typically used to provide feedback, convey information, or alert users about specific events or actions.

## Custom usage

```html
<dp-notification
    v-for="message in messages"
    :key="message.uid"
    :message="message"
    @dp-notify-remove="removeMessage()" />
```
