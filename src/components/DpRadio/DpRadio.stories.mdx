# DpRadio

The DpRadio component is used to render a radio input field with customizable options.

## Examples

export const Template = (args, { argTypes }) => ({
    props: Object.keys(argTypes),
    components: { DpRadio },
    template: `<dp-radio v-bind="$props" />`
});

## Label props

DpRadio passes the `label` prop to the <LinkTo kind="components-label--default-usage">DpLabel component</LinkTo>. It supports the 'bold', 'hint', and 'text' properties.
