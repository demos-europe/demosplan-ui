# DpSelect

The DpSelect component provides `<select>` element to create a drop-down list.

With `label` attribute can select menu title be passed, with `options` attribute can call a computed.

And with `@select` event can call a method.

## Default usage

```html
<dp-select
  v-model="selectModel"
  class="u-mb-0_5"
  label="Select Title"
  name="select_menu_name"
  :options="Computed Name"
  required
  @select="Method Name" />
```
