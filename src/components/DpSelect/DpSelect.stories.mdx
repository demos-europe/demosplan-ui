# DpSelect

The DpSelect component is used to render a select input field with customizable options. It provides functionality for selecting an option from a dropdown list and emits a select event with the selected value.

## Default usage

```html
<dp-select
  v-model="selectModel"
  class="u-mb-0_5"
  label="Select Title"
  name="select_menu_name"
  :options="Computed Name"
  required
  @select="updateLocalState" />
```
