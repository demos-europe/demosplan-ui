# SkeletonBox

The `DpSkeletonBox` component renders an skeleton loading animation with shimmer.

## Default usage

```html
<dp-skeleton-box
    v-for="(item, idx) in items"
    :key="`skeleton:${idx}`"
    height="54px" />
```
