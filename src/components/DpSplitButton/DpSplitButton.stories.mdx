# SplitButton

The SplitButton displays a main action button with an accompanying arrow that reveals more actions in a dropdown. It's useful in situations with limited space, where one primary action stands out and less important actions remain hidden initially.

Dropdown actions may contain links or buttons.

## Default usage

```html
<dp-split-button>
    <a href="#">
        primary action button name
    </a><!--
 --><template v-slot:dropdown>
    <a href="#">
       dropdown link 1
    </a>
    <a href="#">
       dropdown link 2
    </a>
    </template>
</dp-split-button>
```
