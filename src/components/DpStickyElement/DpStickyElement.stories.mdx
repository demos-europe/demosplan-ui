# StickyElement

The StickyElement component is used to create elements that stick to a specific position (top or bottom) while scrolling.

## Default usage

```html
<dp-sticky-element
  direction="bottom">
  <div>
    <!-- Whatever you place here is gonna stick to the bottom -->
  </div>
</dp-sticky-element>
```
