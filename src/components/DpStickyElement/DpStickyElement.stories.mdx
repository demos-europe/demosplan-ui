# StickyElement

The StickyElement component is used to create elements that stick to a specific position (top or bottom) while scrolling.

## Default usage

```html
<dp-sticky-element
  direction="bottom"
  ref="footer"
  class="c-mastertoeb__footer">
  <div class="u-mv-0_5 text-right">
    <sliding-pagination
      :current="currentPage"
      :total="totalPages"
      @page-change="handlePageChange" />
    <dp-select-page-item-count
      @changed-count="setPageItemCount"
      :page-count-options="itemsPerPageOptions"
      :current-item-count="itemsPerPage"
      :translations="{ pagerElementsPerPage: Translator.trans('pager.per.page') }" />
  </div>
</dp-sticky-element>
```
