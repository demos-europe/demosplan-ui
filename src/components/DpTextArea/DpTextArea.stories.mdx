# DpTextArea

The `DpTextArea` component is used to define a multi-line text input control.

## Examples

export const Template = (args, { argTypes }) => ({
    props: Object.keys(argTypes),
    components: { DpTextArea },
    template: `<dp-text-area v-bind="$props" />`
});
