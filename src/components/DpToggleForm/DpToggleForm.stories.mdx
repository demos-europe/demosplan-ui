# ToggleForm

The ToggleForm component is used to toggle the visibility of a forms.

## Default usage

```html
<dp-toggle-form v-bind="args" title="Form Title">
  <form
      action="#"
      method="post">
    <label for="title">Title:</label>
    <input type="text" id="title" name="title"><br><br>
  </form>
</dp-toggle-form>`
```
