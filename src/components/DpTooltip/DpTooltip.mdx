# Tooltip

The Tooltip component displays additional information when a user hovers over or focuses on an element.
It's useful for providing context, explanations, or extra details without cluttering the interface.

## Usage

### Default

Basic tooltip that appears on hover or focus.

## Implementation Notes

The component can be used in two ways:

1. As a wrapper component (shown in these examples)
2. As a directive (`v-tooltip`) that can be applied to any element

The tooltip is positioned using the Floating UI library, which ensures it stays within the viewport and adapts to available space.
