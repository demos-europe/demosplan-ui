# Tooltip Icon

Renders a fontawesome Icon with a tooltip.

This component is deprecated in favor of DpContextualHelp

## Default usage

```html
<dp-tooltip-icon
  icon="fa-trash"
  text="Content of the tooltip"
  large="true|false (adds the class fa-lg)" />
```
