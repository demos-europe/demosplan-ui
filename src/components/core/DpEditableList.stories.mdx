# EditableList

The EditableList component

## Custom usage

```html
<dp-editable-list v-bind="args">
    <template v-slot:list="entry">
        <span>
          {{ entry.mail }}
          <input
              class="hide-visually"
              type="email"
              :value="entry.mail"
              name="formFieldName">
        </span>
    </template>
    <template v-slot:form="entry">
      <input
          id="emailAddress"
          type="email"
          :value="args.entries.mail"/>
    </template>
</dp-editable-list>
```
