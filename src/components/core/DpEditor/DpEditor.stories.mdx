# Editor

A rich text editor component based on tiptap.

- defines a menubar with a number of buttons and an editor

To properly set the content we have to update this.currentValue and use editor.setContent() - both actions are needed!

To use image upload we need to add imageButton optional Prop.

To use another component within DpEditor, we can add Slot named modal.

## Default usage

```html
<dp-editor
    :value="value"
    ref="r_links">
</dp-editor>
```

## Custom usage

```html
<dp-editor
    :toolbar-items="{
        fullscreenButton: true,
        insertAndDelete: true,
        table: true,
        headings: [2,3,4],
        imageButton: true,
        linkButton: true,
        strikethrough: true
    }"
    :value="value">
    <template v-slot:modal="modalProps">
        <dp-modal
          id="openModal"
          ref="openModal">
          Add an component
        </dp-modal>
    </template>
</dp-editor>
```
