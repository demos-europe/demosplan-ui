# TableCard

The TableCard component is used to display tabular data in a visually organised and structured way.

It's often used to present data in a list, making it easier for users to scan and understand the information.

## Custom usage

```html
<ul class="o-list o-list--card u-mb">
  <div class="o-list__item">
    <dp-table-card class="o-accordion u-ph-0_5" v-bind="args">
      <template v-slot:header>
        <div class="flex items-start">
          <div class="relative z-above-zero u-mt-0_75">
              <input
                  type="checkbox">
          </div>
          <button
              @click="args.isOpen = !args.isOpen"
              type="button"
              data-cy="userListItemToggle"
              class="btn--blank o-link--default u-pv-0_75">
            Click to Toggle
          </button>
        </div>
      </template>
      <div>
        Test 1
      </div>
      <div>
        Test 2
      </div>
    </dp-table-card>
  </div>
</ul>
```
