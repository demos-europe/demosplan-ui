# TableCard

The TableCard component is used to display tabular data in a visually organised and structured way.

It's often used to present data in a list, making it easier for users to scan and understand the information.

## Custom usage

```html
<div>
    <ul class="o-list o-list--card u-mb">
      <div class="o-list__item">
        <dp-table-card
            class="o-accordion u-p-0_5"
            ref="card"
            v-bind="args">
            <template v-slot:header>
              <div class="flex">
                <div>
                  <input
                      type="checkbox">
                </div>
                <div class="">
                  <p>Title 1</p>
                </div>
                <button
                    @click="$refs.card.toggle()"
                    type="button"
                    class="btn--blank o-link--default">
                  Click to Toggle
                </button>
              </div>
            </template>
            <template>
              <div>
                Content 1
              </div>
            </template>
        </dp-table-card>
      </div>
    </ul>
</div>
```
