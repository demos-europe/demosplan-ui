# Font Size tokens

The demosplan-ui Font Size tokens variables & font properties to be used within demosPlan.

Global font-size tokens. Do not use them directly.

## How to use and override color tokens

Font Size tokens can be consumed as Scss variables. To use them, they have to be imported at first:

```
@import '@demos-europe/demosplan-ui/tokens/scss/fontSize';
```

Then, tokens can be used within Scss code:

```
.ol-attribution {
    font-size: $dp-font-size-smaller;
}
```

```
.font-size-h1 {
    @include inuit-font-size($dp-font-size-h1, 1.3);
}
```

## Available tokens

### Basic scale values

The numbers count from 0 (smallest possible size) up to 7 (biggest size used). The default font-size is 16px.

```
$dp-font-size-0
// ...
$dp-font-size-7
```

### Brand headings

The `header` font size styles the page header in the participation area.

```
$dp-font-size-header
```

The `hero` font size is used to style the single most gigantic headline of the site.

```
$dp-font-size-hero
```

### Semantic section headings

The `h1` to `h6` sizes are used to create visual hierarchy with heading-like entities that structure long-form content.
Do not use them when styling interface elements. Use the below UI font sizes like `small`, `medium`, or `large` instead.

```
$dp-font-size-h1
...
$dp-font-size-h6
```

### Font sizes for functional interfaces

The `large|medium|small|smaller|smallest` font sizes are used to create hierarchy between interface elements
via font size, like tabs, action buttons and the like.
Do not use it to create visual hierarchy with long text. Go for `h1-h6` tokens instead.

```
$dp-font-size-larger
$dp-font-size-large
$dp-font-size-medium
$dp-font-size-small
$dp-font-size-smaller
$dp-font-size-smallest
```
