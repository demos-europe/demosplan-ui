# Spacing Tokens

Spacing tokens define the consistent spacing scale available throughout demosplan-ui. They are used for padding, margins, gaps, and other spatial relationships in the UI.

## Usage

### Tailwind Usage (Recommended)

Spacing tokens can be used in any Tailwind utility that accepts spacing values:

```html
<!-- Padding -->
<div class="p-4">Padding on all sides</div>
<div class="pt-2 px-4 pb-6">Different padding per side</div>

<!-- Margin -->
<div class="m-2">Margin on all sides</div>
<div class="my-4">Vertical margin</div>

<!-- Width and height -->
<div class="w-12 h-6">Fixed size element</div>

<!-- Gap in flex/grid layouts -->
<div class="flex gap-1.5">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

<!-- The real world example -->
<div className="w-6 h-6 space-y-2 gap-1.5 bg-message-debug">
  Css is awesome.
</div>
```

### SCSS Usage (Legacy)

```scss
// Import spacing tokens
@import '@demos-europe/demosplan-ui/tokens/scss/space';

// Use in your styles
.element {
  padding: $dp-space-4;
  margin-bottom: $dp-space-2;
  gap: $dp-space-1;
  width: $dp-space-14;
}
```
