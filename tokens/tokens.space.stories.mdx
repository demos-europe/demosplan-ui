While the values `0.5` to `6` follow a linear progression with "in-betweens",
the bigger values follow a geometric progression (each step is 1.371 x its predecessor).
This geometric progression aims to reduce the overall number of available steps while
ensuring each step is easily distinguishable from the others.

## Usage example - Scss

Spacing tokens can be consumed as Scss variables. To use them, they have to be imported at first:

```scss
@import '@demos-europe/demosplan-ui/tokens/scss/space';
```

Then, they can be used within Scss code:

```scss
.huge-element {
    width: $dp-space-14;
}
```

## Usage example - Tailwind utility class

As Tailwind is configured with these variables, too, they can be used as tailwind utility classes.
They replace the default classes mentioned in the [Tailwind documentation](https://tailwindcss.com/docs/customizing-spacing).
Instead, the custom spacing scale shown here spacing scale is inherited by the padding, margin, width, height, maxHeight,
gap, inset, space, translate, scrollMargin, and scrollPadding core plugins of Tailwind.
